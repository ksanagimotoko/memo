# memo

### 맥 단축키
> 맥 화면 최대화 토글 cmd + ctrl + f  
강제종료 cmd + opt + esc


### 리눅스
###### 파일보기  
> 파일크기 보기 du -sh *  
목록자세히보기 ls -alh  
###### 압축하기 
> \> tar -cvf [파일명.tar] [압축하려는 폴더명]  
\> tar -xvf [파일명.tar]  
압축 tar cvfz xxx.tar.gz  
압축풀기 tar xvfz  

###### 포트확인  
> 포트 활성화 확인 lsof -i :8085  
포트 목록 보기 lsof -n -P -i TCP -s TCP:LISTEN
